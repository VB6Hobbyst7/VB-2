유로이뮨 장비 이름은 "EUROBlotOne"이라는 장비이고 
결과분석은 " EUROLineScan"이라는 소프트웨어를 사용한다고 합니다.


안녕하세요. 포스트바이오 알러지 검사 담당자 김진선입니다.


기존 검사 5가지에서 새로운 버젼의 검사 4가지로 바뀌게 됩니다.

첨부된 엑셀파일에는 새로 바뀌는 검사들의 검사별 항원을 정리해놓았습니다. 



>> 현 Food Advanced Test

     * 검사명은 Basic Test 로 변경

     * 기존의 항원 11가지 삭제



>> 현 Premium Test

     * 검사명은 동일하게 유지

     * 기존의 항원 11가지 삭제, 유로이뮨의 54가지 항원 추가 



>> 현 Food Intensive Test

     * 검사명은 동일하게 유지 

     * IgE 검사 - 기존의 항원 3가지 삭제, 유로이뮨의 54가지 항원 추가

     * IgG(non_IgE) 검사 - 기존의 항원 3가지 삭제, 유로이뮨의 49가지 항원 추가



>> 현 Premium Intensive Test

     * 검사명은 동일하게 유지 

     * IgE 검사 - 기존의 항원 11가지 삭제, 유로이뮨의 54가지 항원 추가

     * IgG(non_IgE) 검사 - 기존의 항원 3가지 삭제, 유로이뮨의 49가지 항원 추가


>> Inhalant Advanced Test 는 없어집니다.





알러지 검사 장비는 2월이 지나야 들어올 것으로 예상됩니다.

추가되는 유로이뮨의 IgE_54가지 / non_IgE 49가지 항원들은 검사 장비에서 결과값을 읽어들이는 항원의 수와 똑같습니다.

장비가 들어올 때 까지 기다릴려면 검사 셋팅이 너무 늦어져서

우선 프로그램이나 온라인 상으로 가능한 부분까지 먼저 진행하려고합니다.
