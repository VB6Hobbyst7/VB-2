//			예: ~DG0001,__H[굴림||20||__[색상명]__]H__,^FO130,125^XG0001,1,1^FS
//  수식 Binding : 
//      포맷 : "__{" + 수식 + "}__"
//      예제 : __{1+2+3}__ => 6, __{__[가격]__*2}__ => 가격컬럼값*2
//      연산 : +,-,*연산만가능
//      함수 : ROUND(), 기타 미확인.... 
//  포맷함수 Binding : vb의 Format(,) 함수가 이용됨. 
//			포맷 : "__F(" + 수치데이터 + || + 포맷문자열 + ")F__"
//      예제 : __F(12345.678||#,##0.00)F__ => 12,345.67

//      _HOME_X_,_HOME_Y_ : 홈좌표용
//================================================================================================================
