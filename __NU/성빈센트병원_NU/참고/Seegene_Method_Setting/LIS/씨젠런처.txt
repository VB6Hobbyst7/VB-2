씨젠 런처에서 interface하여 데이터를 불러올 때 사용되는 파일입니다.
경로는 C:\Seegene_Method_Setting 이며 안에 "LIS"와 "CFX96" 2개 폴더가 있습니다.

LIS 폴더에는 LIS.lis 라는 파일이 한개 존재하며 기능 실행 시 해당 파일이 바로 불러와 집니다.
폴더 안에 파일 형식이 정리된 문서도 같이 첨부되어 있습니다.

CFX96 폴더에는 임의의 파일명으로 csv 파일이 존재하며 기능 실행 시 해당 폴더 안의 파일을 선택할 수 있습니다.
csv 파일 column 기준은  Row,Column,*Target Name,*Sample Name 으로 되어 있습니다.
샘플 리스트 정보는 제품별로 다를 수도 있기 때문에 별도의 파일 형식을 정리할 수 없지만 파일 열어보시면 이해하실 수 있을 거 같습니다.
일단 작업하시는 곳은 HPV28 제품이라고 알고 있는데요, HPV28 제품 기준으로 A01, B01, C01...H01, A02, B02 순으로 H05까지 샘플 입력이 가능합니다.
A07~H12까지의 정보는 A01~H05 와 동일 샘플 정보이니 참고하시면 될 거 같습니다.

LIS 폴더에 있는 파일은 프로그램 상에서 "Load LIS" 기능을 사용하면 처리되고요,
CFX96 폴더에 있는 파일은 프로그램 상에서 "Load File" 기능을 사용하면 처리됩니다.
